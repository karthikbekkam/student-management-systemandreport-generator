Student Management System & Report Generator

Description:
A simple application used to manage student records and generate reports. The system allows adding, viewing, updating, and deleting student details using file handling techniques. This project is suitable as a university mini project.

Technologies Used:
C / Python
File Handling

Features:
Add student details
View student records
Update student information
Delete student data
Generate student reports

How to Run:

C Version:
gcc main.c -o student
./student

Python Version:
python main.py

Project Use:
University mini project
Practice of file handling and CRUD operations
